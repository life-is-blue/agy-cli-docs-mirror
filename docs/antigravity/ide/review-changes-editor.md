# Review Changes

Once the agent has begun writing code within a conversation, you’ll see a `Review Changes` section within the Agent panel’s [bottom toolbar](docs/ide/agent-side-panel). Clicking it will open up a pane within your editor where you can scroll through all of the changes you and your agent made within the conversation. 

![Editor Review Changes](assets/image/docs/editor/review_changes_editor.png)

Just like with artifacts, you can comment on any of the file diffs to communicate with the agent.
